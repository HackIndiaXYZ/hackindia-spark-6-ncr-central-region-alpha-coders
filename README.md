## 📌 About The Project

Traditional coding platforms can feel solitary and monotonous. **CodeArena AI** revolutionizes how developers learn and practice by merging the intense, adrenaline-pumping aesthetics of AAA Battle Royale games with competitive programming. 

Whether you are warming up in the *Training Grounds* or battling head-to-head in *Ranked Matchmaking*, CodeArena provides a seamless, real-time multiplayer experience powered by WebSockets and beautiful React animations.

## ✨ Key Features

- **🎮 Immersive Gamified UI**: Built from the ground up with neon scanlines, 3D rotating HUDs, and cinematic animations that make you feel like you are hacking a mainframe.
- **⚔️ Live Multiplayer Battles**: Real-time matchmaking connects you with developers instantly using WebSockets.
- **🎯 Training Grounds**: A curated database of 45+ real algorithmic missions divided into Sectors (Arrays, Strings, Loops, Functions) with calculated XP yields.
- **🤖 AI Opponents**: Can't find a match? Train against dynamic AI bots of varying ranks (Platinum, Diamond, Grandmaster).
- **🏆 Global Leaderboard & Ranks**: Earn XP, complete Daily Ops, and climb the tiers to establish your dominance.

## 🛠 Tech Stack

**Client / Frontend:**
- [React.js](https://reactjs.org/) (Vite)
- [Framer Motion](https://www.framer.com/motion/) (Cinematic page transitions and physics-based animations)
- [Tailwind CSS](https://tailwindcss.com/) (Custom neon variants, dynamic gradients, 3D utilities)
- [Lucide Icons](https://lucide.dev/)
- Monaco Editor (VS-Code style embedded compiler mapping)

**Server / Backend:**
- [Node.js](https://nodejs.org/) & [Express.js](https://expressjs.com/)
- [Socket.io](https://socket.io/) (Bi-directional real-time communication protocols)

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites
Make sure you have Node.js and npm installed on your machine.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/HackIndiaXYZ/hackindia-spark-6-ncr-central-region-alpha-coders.git
   ```
2. **Start the Backend Server**
   ```sh
   cd backend
   npm install
   node server.js
   ```
3. **Start the Frontend Client** (in a new terminal)
   ```sh
   cd frontend
   npm install
   npm run dev
   ```
4. Open your browser to `http://localhost:5173` and start your mission!
